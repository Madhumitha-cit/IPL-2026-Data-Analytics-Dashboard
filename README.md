# IPL 2026 Data Analytics Dashboard

## Overview
This project is an interactive IPL 2026 Data Analytics Dashboard built using Python, Pandas, Plotly, and Streamlit. It analyzes ball-by-ball IPL data and presents insights through interactive visualizations.

## Features
- Top Run Scorers
- Top Six Hitters
- Top Wicket Takers
- Highest Individual Scores
- Most Fours
- Most Dot Balls
- Best Bowling Figures
- Highest Team Scores
- Venue Analysis

## Technologies Used
- Python
- Pandas
- Plotly
- Streamlit

## Project Structure
```
IPL_2026_Data_Analytics_Dashboard/
│
├── app.py
├── requirements.txt
├── data/
│   └── ipl_2026_deliveries.csv

```

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
